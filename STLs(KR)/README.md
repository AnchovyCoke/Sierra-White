## KlickyProbe

역시 국내 규격 프로파일에 대한 호환성을 높였다. 또한 기존 모델링은 ABS 재료를 사용할 시 실제 출력물이 수축되며 3x6mm 네오디뮴 자석을 부착할 수 없었다. 따라서 자석이 들어가는 구멍과 배선이 통과할 수 있는 부분을 수축공차를 고려하여 수정했다.
